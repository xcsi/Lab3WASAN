## Dataset
Amazon-Style E-Commerce Sales Dataset (Synthetic)

## Objectives
This project applies basic Exploratory Data Analysis (EDA) techniques:
- Data structure examination
- Handling missing values
- Removing duplicates
- Data visualization
- Correlation analysis
- Time trend analysis
- Extracting insights

## Files Included
- lab3.ipynb → Jupyter Notebook containing the full EDA process
- amazon_sales_dataset.csv → Dataset used for analysis

## Key Findings
- The dataset contains 10,000 rows and 21 columns.
- No missing values or duplicate records were found.
- Total sales are strongly influenced by unit price and quantity.
- Discount shows a weak negative relationship with total sales.
- Monthly sales trends show variation over time.

This analysis demonstrates practical application of EDA techniques.
